# Short Film ポストプロダクション

生成したクリップ・音声をDaVinci Resolve / Premiere Proで組み立てて最終作品にするフェーズ。
Claudeは編集ソフトを直接操作できないため、**ステップバイステップの指示**を提供する。

会話は**日本語**。

---

## 【必須】Step 0: プロジェクト確認

1. `projects/shortfilm/[film-name]/brief.md` と `asset-checklist.md` を確認
2. 全アセットがダウンロード済みか確認

ユーザーに確認:
```
1. 編集ソフト: DaVinci Resolve / Premiere Pro / その他
2. 全クリップ・音声はローカルに保存済み？
3. 書き出し先プラットフォーム: YouTube / TikTok / Instagram / 映画祭
```

---

## Step 1: プロジェクトセットアップ

### フォルダ構成

```
[film-name]/edit/
├── footage/           # クリップ（シーン番号-ショット番号で命名）
│   ├── S01-01.mp4
│   ├── S01-02.mp4
│   └── ...
├── broll/             # B-Rollクリップ
├── audio/
│   ├── dialogue-char1.wav
│   ├── dialogue-char2.wav
│   ├── bgm.wav
│   └── sfx/
├── images/            # Hero Frames（参照用）
└── exports/           # 最終書き出し
```

### タイムライン設定

| 設定 | YouTube 16:9 | TikTok 9:16 | Cinema 21:9 |
|------|-------------|-------------|-------------|
| 解像度 | 1920x1080 / 3840x2160 | 1080x1920 | 2560x1080 |
| FPS | 24fps（映画的）| 30fps | 24fps |
| カラースペース | Rec.709 | Rec.709 | Rec.709 |

---

## Step 2: クリップ配置

### 基本手順

1. **ショットリスト順にクリップをタイムラインに配置**
2. Scene 1 → Scene 2 → Scene 3 の順序
3. 各シーン間にマーカーを打つ

### カット編集

- **不要部分をトリム:** 生成ミス、意図しない動き、長すぎるシーン
- **Cinema Studioの最初の数フレーム**は不安定なことがある → カット
- **ミラーリング:** 構図が逆の方が良い場合はクリップを左右反転

### Tips

- 生成結果に**すべてが完璧なショットは少ない** — 使えるフレームだけ抽出
- 同じショットを複数回生成して**ベストテイクを選ぶ**のが前提
- 足りないショットに気づいたら → `/shortfilm-prod` で追加生成（同じワークフローで簡単に追加可能）

---

## Step 3: B-Roll インサート

対話シーンの合間にB-Rollを挿入して**シーンに呼吸を与える**:

### 配置パターン

| パターン | 効果 |
|----------|------|
| 会話の合間にカットアウェイ | テンポ調整、視覚的変化 |
| 音声を続けながらB-Roll映像 | ナレーション感、情報量UP |
| シーン転換にB-Roll | 時間/場所の変化を示す |

### 音声オーバーラップ

対話音声をB-Roll映像の上に乗せる:
- 例: キャラの声が聞こえるが、映像はダッシュカムPOVの街並み
- → 臨場感と映画的な編集感

---

## Step 4: オーバーレイ（映画感UP）

**これがAI動画を映画的に見せる最大の秘密。**

### Luma Key / Gaussian Blur オーバーレイ

1. 同じクリップをV2トラックにコピー
2. **ブレンドモード:** Screen または Add
3. **Gaussian Blur（ガウスぼかし）** を強めに適用（20-40px）
4. **不透明度:** 10-25%に下げる

**効果:** ドリーミー、ソフトグロー、シネマティックな質感。レンズフレアに近い印象。

### DaVinci Resolveの場合

1. Edit Page → V2にクリップ複製
2. Inspector → Composite → Blend Mode: Screen
3. Color Page → Blur → Radius: 25-35
4. Composite → Opacity: 15-20%

### Premiere Proの場合

1. V2にクリップ複製
2. エフェクト → ブラー → ガウスぼかし（25-35）
3. 不透明度 → ブレンドモード: スクリーン
4. 不透明度: 15-20%

## Step 5: カラーグレーディング

### シーンごとの色調ガイド

| 感情 | 色温度 | サチュレーション | コントラスト | 参考 |
|------|--------|-----------------|------------|------|
| 孤独/悲しみ | 冷色（青寄り） | やや低め | 高め | ブレードランナー |
| 暖かさ/安心 | 暖色（オレンジ寄り） | 標準 | 標準 | アメリ |
| 緊張/サスペンス | 脱色気味 | 低い | 高い | セブン |
| 日常/ドキュメンタリー | ニュートラル | やや高め | 低め | ダッシュカム感 |
| 喜び/希望 | 暖色 | やや高め | 標準 | — |

### Cinema Studio生成時に色調を統一済みの場合

軽いLUT適用とコントラスト微調整のみでOK。過度な補正は避ける。

## Step 6: 音声ミックス

### トラック配置

```
A1: キャラクター1 対話
A2: キャラクター2 対話
A3: BGM
A4: SFX / 環境音
```

### レベルバランス（目安）

| トラック | dB |
|----------|-----|
| 対話 | -6 〜 -12 dB |
| BGM（対話あり時） | -18 〜 -24 dB |
| BGM（対話なし時） | -12 〜 -16 dB |
| SFX | -12 〜 -18 dB |

### Tips

- 対話が最優先 — BGMは対話時に自動ダッキング設定
- シーン転換でBGMのクロスフェード
- 環境音（エンジン音、ラジオ等）は低く一定に

---

## Step 7: エクスポート設定

### YouTube (16:9)

| 設定 | 値 |
|------|-----|
| コーデック | H.264 / H.265 |
| 解像度 | 1920x1080 or 3840x2160 |
| FPS | 24 |
| ビットレート | 20-50 Mbps (1080p) / 50-100 Mbps (4K) |
| オーディオ | AAC 320kbps |

### TikTok / Instagram Reels (9:16)

| 設定 | 値 |
|------|-----|
| コーデック | H.264 |
| 解像度 | 1080x1920 |
| FPS | 30 |
| ビットレート | 15-25 Mbps |

### 映画祭 / 高品質 (21:9)

| 設定 | 値 |
|------|-----|
| コーデック | ProRes 422 or DNxHR |
| 解像度 | 2560x1080+ |
| FPS | 24 |

---

## Step 8: 最終レビューチェックリスト

- [ ] 全シーンの繋がりに違和感がないか
- [ ] 対話のタイミングと口の動きが合っているか（完璧でなくてOK）
- [ ] B-Rollのインサートが自然か
- [ ] オーバーレイが効きすぎていないか
- [ ] カラーグレーディングがシーンのムードに合っているか
- [ ] 音声レベルが一定か（急な音量変化がないか）
- [ ] 冒頭2秒で視聴者を引きつけるフックがあるか
- [ ] エンディングに余韻があるか
- [ ] テクニカル: 正しい解像度・FPS・アスペクト比でエクスポートしたか
