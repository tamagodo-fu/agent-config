# Short Film プリプロダクション

Higgsfield AIでショートフィルムを作るための企画フェーズ。
ストーリー構想からキャラクター設計、ショットプランまでを体系的に進める。

会話は**日本語**、生成プロンプトは**英語**で出力。

---

## 【必須】Step 0: プロジェクト確認

スキル発動時、AskUserQuestionで確認:

```
1. 新規プロジェクト or 既存プロジェクト続行？
2. ジャンル: ドラマ / SF / ホラー / コメディ / ドキュメンタリー / その他
3. プラットフォーム: TikTok(9:16) / YouTube(16:9) / Cinema(21:9) / Instagram(1:1, 4:5)
4. 尺: 30秒 / 1分 / 3分 / 5分+
```

新規の場合 → `projects/shortfilm/[film-name]/` ディレクトリを作成。
`templates/project-brief.md` をコピーして `brief.md` として配置。

---

## Step 1: 感情から始める（ストーリーの核）

**プロンプトではなく「感情」から始める。**

ユーザーに問いかける:
- 視聴者にどんな感情を感じさせたい？
- あなたが最近感じた強い感情は？
- 映画で泣いた/鳥肌が立った瞬間は？

### 3文プレミスの作成

フォーマット:
```
1. キャラクター: 誰が？どんな状況の人？
2. 場所/状況: どこで？何が起きている？
3. 決定的瞬間: 何が変わる？どうなる？
```

例（Dan Kieft参照）:
```
1. 鬱を抱える警官。
2. 誕生日だが、本人は何も特別なことはないと言い張る。
3. 同僚と妻がサプライズパーティーを仕掛ける。
```

---

## Step 2: 3シーン構成

短編映画は**3シーンで十分**:

| シーン | 役割 | 例 |
|--------|------|-----|
| Scene 1 | セットアップ | ロッカールーム。静か、親密。相棒が異変に気づく |
| Scene 2 | ミッドポイント | パトカー。カジュアル、少し緊張。相棒が励まそうとする |
| Scene 3 | クライマックス | サプライズ。すべてが変わる瞬間 |

各シーンに設定する項目:
- **場所** — 具体的なロケーション
- **感情トーン** — そのシーンのムード
- **カメラスタイル** — シーンごとに変えると映画的（例: Scene 1はシネマティック、Scene 2はダッシュカム風）
- **カラーパレット** — 赤=激情、青=孤独/冷静、暖色=安心
- **対話の有無** — あれば概要

---

## Step 3: ムードボード

ユーザーにムードボード作成を促す:

### 収集先
- **Pinterest** — 構図、照明、色調のリファレンス
- **YouTube** — 映画のスクリーンショット
- **Instagram** — 写真家の作品
- **映画** — 参考にしたい作品名を聞く

### シネマティック画像の4要素

Hero Frame（静止画）の品質を決める要素:

1. **ライティング** — 方向性のある光。フラットNG。影・コントラスト・ハイライト・深度を生む
2. **構図** — Rule of Thirds（三分割法）。被写体をグリッド線/交点に配置。「slightly off-centered」でプロンプト
3. **深度** — 前景・中景・背景を作る。フラットNG
4. **色** — 感情を伝える色選択。プロンプトで色調変更可能

---

## Step 4: キャラクター設計

### Soul ID トレーニング（キャラクター一貫性の鍵）

**20枚以上の写真が推奨。** 以下をカバー:

| カテゴリ | 内容 |
|----------|------|
| 正面 | 均一照明、ニュートラル表情 |
| 斜め45° | 左右両方 |
| 横顔 | 左右 |
| クローズアップ | 顔のディテール |
| 全身 | 立ちポーズ |
| 全身（左向き） | 体の向き変化 |
| 全身（右向き） | 体の向き変化 |
| 異なる表情 | 笑顔、真顔、悲しみ等 |
| 異なる照明 | 自然光、室内光、暗所 |

**実在人物がいない場合:**
Nano Banana 2で20枚のバリエーション画像を生成 → Soul IDにアップロード。
→ プロンプト生成は `/higgsfield` スキルに委譲。

**【重要】Soul IDのモデル選択:**
- Soul IDトレーニング時に**モデルを選択**する必要がある
- **Soul Cinema**で使うなら → Soul Cinemaモデルを選択（Hero Frame生成用）
- **Soul 2.0**は別モデル。Soul 2.0で作ったキャラはSoul Cinemaでは「Recreate」が必要（追加クレジット消費）
- **最初からSoul Cinemaモデルで作成**すれば二度手間にならない

**【重要】リファレンス画像による一貫性確保:**
1. まず**プロンプト1（正面ニュートラル）だけ**を生成する
2. 結果が気に入ったら、その画像をダウンロード
3. 以降のプロンプト2〜20は、**正面ニュートラル画像をリファレンスとしてアップロード**した上でプロンプトを入力
4. これにより顔の骨格・特徴が全バリエーションで維持される
5. プロンプトmdファイルには、この手順を冒頭に明記すること

### キャラクターシート生成

Soul ID登録後、Nano Banana Proでキャラクターシートを生成:
- 正面・横・斜め・背面のターンアラウンド
- 衣装・アクセサリー・表情のバリエーション
- → `/higgsfield` でプロンプト生成

### プロップシート / ロケーションシート

シーンの環境一貫性のために:
- ロケーション全体の参照画像を生成（Soul Cinemaで）
- 重要な小道具（例: ボトル、マグカップ）の参照画像
- → 全ショットでこれらをElement Referenceとして使用

---

## Step 5: ショットリスト作成

`templates/shot-list.md` を基に、各シーンのショットを計画:

各ショットに:
- **ショットタイプ:** Wide / Medium / Close-up / Extreme CU / OTS / POV
- **カメラムーブ:** Static / Handheld / Dolly / Truck / Crane / FPV
- **尺:** 3-7秒（Cinema Studio 2.0の推奨）
- **被写体:** 誰/何が映るか
- **アクション:** 何が起きるか
- **対話:** あれば概要

### B-Rollの計画

本編の合間に入れるカットアウェイ:
- ダッシュカムPOV
- 小道具のクローズアップ
- 環境ショット（街並み、窓の外等）
- → Soul Cinemaでシンプルなプロンプトで生成可能

---

## Step 6: ブリーフ出力

すべてを `[film-name]/brief.md` に記録。

ブリーフ完成後:
```
次のステップ → /shortfilm-prod でプロダクション開始
```
